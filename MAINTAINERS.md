## Becoming a Maintainer

ContextDuty needs additional maintainers to be viable for enterprise use.

If you are using ContextDuty in production, have made meaningful contributions,
and want to take on a maintainer role, open a GitHub Discussion. Maintainer
responsibilities:

- Review and merge pull requests within a reasonable timeframe
- Triage issues and respond to security reports
- Participate in release decisions
- Uphold the Code of Conduct
